# GDApps Studio Apps

Monorepo powered by pnpm and Turborepo.

## Apps

- **merkle-tree-generator** — Generates Merkle trees for whitelist-based airdrops on Ethereum and Solana.
- **donation** — Crypto donation flow with EVM and Solana wallet support.
- **landing-page** — Marketing site with 3D visuals, animations, and a contact form.
- **onchain-payment** — Crypto payment form supporting Ethereum and Solana.

## Packages

- **ui** — Shared React UI component library (button, card, dialog, form, etc.).
- **payment-card** — Reusable crypto payment card component.
- **assets** — Shared assets (logos, social icons).
- **eslint-config** — Shared ESLint configurations.
- **typescript-config** — Shared TypeScript configurations.
